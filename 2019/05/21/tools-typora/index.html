Typora

简介

  • Typora 删除了预览窗口，取而代之的是实时预览
  • Markdown的语法因不同的解析器或编辑器而异，Typora使用的是GitHub Flavored Markdown。

常用快捷键

  • 加粗: Ctrl + B
  • 标题: Ctrl + 0/1/2/..
  • 插入链接：Ctrl + k
  • 行内代码: Ctrl + Shift + `
  • 有序列表： Ctrl + Shift +]
  • 无序列表：Ctrl + Shift + [
  • 插入图片：Ctrl + Shift +I
  • 撤销: Ctrl + z
  • 公式快：Ctrl + Shift + M
  • 代码块：Ctrl + Shift + K
